# ERPlite
A lightweight ERP system built for small precision manufacturing operations.
Built with Flask (Python) and PostgreSQL, ERPlite replaces paper-based job tracking with a web-based system for managing work orders, parts, customers, suppliers, inspections, and shop floor activity -- all behind role-based user authentication.
Modules: Work Orders, Shop Floor, Inspections (NCR tracking), Parts, Customers, Suppliers, Reports, User Management
Stack: Python, Flask, PostgreSQL, HTML/CSS
